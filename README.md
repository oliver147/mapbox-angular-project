# Mapbox Angular Project

This is our starter project for the hackathon. This is an Angular version 17.3.0 project.

## Inital Hackathon Setup

Mapbox, Turf and Angular Material are installed. Mapbox token is configured. Base Layer chooser is implemented and some basic GeoJson Features are put on the map and clickable.

## Development server

Run `ng serve` for a dev server. Navigate to `http://localhost:4200/`.
